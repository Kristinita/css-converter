Thank you for installing the plugin!

Please submit an issue on github if you have any ideas to improve the plugin.
https://github.com/lnikell/css-converter/issues

You can easily configure it with the following settings:

{
  "indent": 2,
  "colon": true,
  "eol": "",
}

Default hotkey is "ctrl + v", but you could change it to another:

[
  { "keys": ["ctrl+v"], "command": "css_to_sass" },
]
